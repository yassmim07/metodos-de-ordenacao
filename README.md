# Métodos de Ordenação
Atividade realizada para a matéria de Algoritmos e Programação II.

Para rodar, baixe o projeto, dizipe a pasta e abra o projeto no Apache NetBeans. 
Dendro do pacote "ado3" vá na classe "Principal_ADO3" e aperte em "Run file". 

O tamanho de cada volume de dado dentro do vetor é pré-definido. Caso queira mudar, vá na pasta "comandos_principais" -> "gerarRandom" e altere para os valores que preferir.
